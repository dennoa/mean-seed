swagger

This project provides support for a Swagger API.

The main server/swagger.js file includes the various paths and defintions from each api. For example, it includes the server/api/clientConfig/swagger.js file. The intent is that each api defines its own swagger docs.

See the OpenAPI Specification for details on how to describe your api.

To test out the API, you can use the Swagger UI. Just point it to your own /swagger.
