MEAN Seed
{{env}} environment

This project can be used as a starting point for building MEAN stack applications.

  1. Copy everything to your new project folder
  2. Change all occurrences of meanSeed to your own angular module name. Note that this name is also used in the tasks/build.js and package.json files
  3. Change all occurrences of mean-seed to reflect your own application name. This is used for mongo database names in the server/config/environment config files
  4. Change the MEAN Seed application name configured in client/index.html and server/swagger.js to your own application name
  5. If you need a database, uncomment the mongoose.connect statement in server/server.js
  6. If you don't need a database, remove the mongoose code from server/server.js, remove the config from the various server/config/environment files and remove the mongoose module dependency from package.json
  7. Get dependencies with npm install
